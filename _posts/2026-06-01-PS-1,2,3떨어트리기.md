## 1. 해법 추론

### 1-1. Drops 횟수 최대/최소 계산

우선 각 리프 노드마다 몇 번의 숫자가 떨어져야 target에 도달할 수 있는지를 알아봐야 한다.

떨어뜨리는 숫자는 1, 2, 3으로 제한되어 있기 때문에 1만 떨어뜨리면 시행의 최댓값을 얻을 수 있고 큰 수 위주로 떨어뜨리면 시행의 최솟값을 얻을 수 있다.

따라서 각 리프 노드에 대해서 구간$$[(\mbox{target}[i] - 1)/3 + 1, \mbox{target}[i]]$$내에 모든 시행이 완료되야 함을 알 수 있다.

>여기서 사용하는 나눗셈 기호 $$/$$는 몫을 계산한다는 것을 잊지 말자.
{: .prompt-info}

그리고 이 각각의 리프 노드에 대해서 최솟값 $$m_i\equiv (\mbox{target}[i] - 1)/3 + 1$$ 최댓값 $$M_i \equiv \mbox{target}[i]$$라고 하자.

<hr/>

### 1-2. 각 리프 노드에 대해서 drops 주기 구하기

위에서 drops의 최대 최소를 구했으니 각 리프에 대해서 몇 회차 시도 때마다 숫자가 떨어지는지를 알아내면 각 리프에 대해서 최종 시행의 범위를 얻어낼 수 있고 target에 도달할 수 있음의 여부를 판단할 수 있게 된다. 

일단 이 판단의 이야기는 뒤에서 서술하도록 하고 우선 각 리프에 대해서 숫자가 떨어지는 규칙을 찾아보도록 하자.

문제에서 제시하고 있는 예시를 기준으로 설명해보자면, 

> 노드 4에 숫자가 떨어지는 회차는 1, 3, 5, 7...이다.

> 노드 9에 숫자가 떨어지는 회차는 4, 10, 16, 22...이다.

대충 모든 노드가 등차수열의 형태를 띈다는 것을 알 수 있다.

이 발견을 모든 노드에 대해서 체계적으로 계산하는 방법을 유도해보자. 

<hr/>

우선 루트 노드인 1번 노드는 모든 회차에서 숫자가 지나가는 것을 알 수 있다. 따라서 자연수 $$n$$에 대해서 $$n$$으로 그냥 둘 수 있다.

그리고 루트의 자식인 2번 노드는 **1번(루트)에서 내려온 숫자** 중 1, 3, 5...번째이고 3번 노드는 2, 4, 6...번째 회차 때 숫자가 지나간다.

>여기서 중요하게 봐야 하는 것이 바로 부모에서 내려온 숫자들 중 1, 3, 5... 그리고 2, 4, 6...번째 라는 것이다.
{: .prompt-warning}

그리고 다시 4번은 2번에서 내려온 숫자 중 1, 3, 5...번째, 5번은 2, 4, 6... 번째 회차 때 숫자가 지나간다.(=떨어진다.)

그리고 7번은 5번에서 내려온 모든 숫자들이 그대로 떨어진다.

그리고 6번은 3번에서 내려온 모든 숫자들이 그대로 떨어진다.

8번 9번 10번은 각각 6에서 내려온 숫자 중 1, 4, 7... / 2, 5, 8... / 3, 6, 9... 번째 회차 때 숫자가 떨어진다.

이를 공식화 해보면 아래와 같다.

>어떤 노드에 숫자가 떨어지는 회차를 $$f(n)$$이라고 하고 그 자식의 수가 $$k$$라고 하면 $$i(1\leq i \leq k)$$번째 자식에 숫자가 떨어지는 회차는 $$f(kn-k+i)$$가 된다.
>그리고 이렇게 얻어낸 $$f(n)$$은 등차수열의 형태이기 때문에 $$f(n)$$을 $$a_{i0}, d_i$$로 표현할 수 있다.
{: .prompt-tip}

<hr/>

### 1-3. 각 리프 노드에 대해서 가능한 전체 시행 횟수의 범위 유도

아까 위에서 drops의 최대 최소를 구했으니 각 리프에 대해서 몇 회차 시도 때마다 숫자가 떨어지는지를 알아내면 각 리프에 대해서 최종 시행의 범위를 얻어낼 수 있고 target에 도달할 수 있음의 여부를 판단할 수 있게 된다고 했다.

우리는 지금까지 각 리프 노드에 대해서 $$m_i,\ M_i,\ a_{i0},\ d_i $$를 얻어냈다.

이 숫자들을 기반으로 각 리프 노드에 대해서 목표가 달성 가능한 시행의 최종적인 범위를 계산할 수 있고 그 값은 아래와 같다.

>$$i$$번 노드에 대해서 목표가 달성 가능한 시행의 범위는 구간$$[a_{i0}+m_id_i,\ a_{i0}+(M_i+1)d_i-1]$$이다.
>
>왜냐하면 적어도 $$m_i$$번째 숫자 drops가 발생해야지 target에 도달할 수가 있고 아무리 회차가 늦어져도 $$M_i+1$$번째 숫자가 도달하기 전에는 시행이 끝나야 하기 때문이다.
{: .prompt-tip}

각 리프 노드에 대해서 이 모든 구간을 계산하고 겹치는 구간을 얻어내면 그 구간이 target에 도달할 수 있는 시행 횟수 구간이 될 것이다.

>이때 공통되는 구간이 없을 수 있는데 이 경우 문제에서 제시한 대로 $$[-1]$$을 반환하면 된다.
{: .prompt-warning}

그리고 그 구간의 최솟값이 문제에서 원하는 최소 시행 횟수(숫자를 가장 적게 쓰는 경우)가 될 것이다. 

그리고 이 최소 시행 횟수를 $$t$$라고 두자.

<hr/>

### 1-4. 최소 시행 횟수 $$t$$에 대해서 각 리프 노드에 떨어뜨려야 하는 숫자 역산 그리고 순서 구하기와 최종 답 유도

t를 알고 있고 각 리프 노드에 대해서 $$a_{i0},\ d_i$$를 알고 있으니 각 리프 노드에 대해서 몇 번의 drops가 발생하는지를 계산할 수 있다. 

$$a_{i0}+d_i(u-1)\leq t$$를 만족시키는 $$u$$의 최댓값을 찾으면 될 것이다.

따라서 $$u\leq (t-a_{i0})/d_i+1$$를 계산해보면 알 것이다.

횟수를 알아냈으니 어떤 숫자를 어떤 방식으로 나열해야하는 지를 알아내야 하는데 문제에서 사전 순으로 빠른 경우를 채택하라고 했기 때문에 전부 1로 두고 뒤에서부터 큰 수로 채워가면 될 것이다.

>이게 가능한 이유는 각 리프 노드의 나열 방식은 다른 리프 노드의 나열 방식과 독립적이기 때문이다.
{: .prompt-info}

## 2. 코드(C++)

필자는 C++로 PS를 하는 것이 편해 C++로 하지만 처음 시작하는 거라면 python을 추천한다.

```cpp
#include <string>
#include <vector>
#include <iostream>
#include <algorithm>
#include <utility>
#include <queue>
#define pii pair<int, int>

using namespace std;

vector<int> solution(vector<vector<int>> edges, vector<int> target) {
    vector<int> answer;

    vector<vector<int>> adjacent_matrix;
    vector<pii> node_cycles;
    vector<pii> satisfy_to_target_min_max;
    vector<int> leaf_node_index;
    vector<int> number_of_drops_by_leaf;
    vector<int> target_copy;
    int number_of_nodes = 0;

    for (int i = 0; i < target.size(); i++) {
        target_copy.push_back(target[i]);
    }
    for (int i = 0; i < edges.size(); i++) {
        if (edges[i][0] > number_of_nodes) {
            number_of_nodes = edges[i][0];
        }
        if (edges[i][1] > number_of_nodes) {
            number_of_nodes = edges[i][1];
        }
    }
    adjacent_matrix.resize(number_of_nodes + 1);
    node_cycles.resize(number_of_nodes + 1, { 0,0 });
    satisfy_to_target_min_max.resize(number_of_nodes + 1, { -1,-1 });
    for (int i = 0; i < edges.size(); i++) {
        adjacent_matrix[edges[i][0]].push_back(edges[i][1]);
    }
    for (int i = 1; i < adjacent_matrix.size(); i++) {
        sort(adjacent_matrix[i].begin(), adjacent_matrix[i].end());
        if (adjacent_matrix[i].size() == 0) {
            leaf_node_index.push_back(i);
            number_of_drops_by_leaf.push_back(0);
        }
    }

    node_cycles[1] = make_pair(1, 1);
    queue<int> queue_for_tree_traversal;
    queue_for_tree_traversal.push(1);

    while (!queue_for_tree_traversal.empty()) {
        int node = queue_for_tree_traversal.front();
        queue_for_tree_traversal.pop();

        for (int i = 0; i < adjacent_matrix[node].size(); i++) {
            int child_node = adjacent_matrix[node][i];
            queue_for_tree_traversal.push(child_node);
            node_cycles[child_node] = make_pair(node_cycles[node].first + node_cycles[node].second * i, node_cycles[node].second * adjacent_matrix[node].size());
        }
    }

    for (int i = 0; i < leaf_node_index.size(); i++) {
        int index = leaf_node_index[i];
        if (target[index - 1] != 0) {
            satisfy_to_target_min_max[index] = make_pair((target[index - 1] - 1) / 3 + 1, target[index - 1]);
        }
        else {
            satisfy_to_target_min_max[index] = { 0, 0 };
        }
    }

    for (int i = 0; i < leaf_node_index.size(); i++) {
        int node_index = leaf_node_index[i];
        pii min_max_temp = satisfy_to_target_min_max[node_index];
        pii cycles_temp = node_cycles[node_index];
        satisfy_to_target_min_max[node_index] = make_pair(
            cycles_temp.first + (min_max_temp.first - 1) * cycles_temp.second,
            cycles_temp.first + min_max_temp.second * cycles_temp.second - 1);
    }

    pii common_interval = make_pair(-1, 10000000);
    for (int i = 0; i < leaf_node_index.size(); i++) {
        pii sub_interval = satisfy_to_target_min_max[leaf_node_index[i]];

        if (sub_interval.first > common_interval.first) {
            common_interval.first = sub_interval.first;
        }
        if (sub_interval.second < common_interval.second) {
            common_interval.second = sub_interval.second;
        }
    }

    if (common_interval.first > common_interval.second) {
        return { -1 };
    }

    answer.resize(common_interval.first, 0);

    for (int i = 0; i < leaf_node_index.size(); i++) {
        pii temp = node_cycles[leaf_node_index[i]];
        if (common_interval.first >= temp.first) {
            number_of_drops_by_leaf[i] = (common_interval.first - temp.first) / temp.second + 1;
        }
    }

    for (int i = 0; i < leaf_node_index.size(); i++) {
        int node_index = leaf_node_index[i];
        for (int j = number_of_drops_by_leaf[i] - 1; j >= 0; j--) {
            int drop_index = node_cycles[node_index].first + j * node_cycles[node_index].second;

            if (target_copy[node_index - 1] - 3 >= j) {
                target_copy[node_index - 1] -= 3;
                answer[drop_index - 1] = 3;
            }
            else if (target_copy[node_index - 1] - 2 == j) {
                target_copy[node_index - 1] -= 2;
                answer[drop_index - 1] = 2;
            }
            else {
                target_copy[node_index - 1] -= 1;
                answer[drop_index - 1] = 1;
            }
        }
    }

    return answer;
}
```
